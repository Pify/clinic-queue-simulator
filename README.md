# Clinic Queue Simulator

A lightweight TypeScript project for simulating patient queues and estimating wait times based on doctor availability.

## Features

- Assigns each patient to the doctor who becomes free the earliest.
- Calculates estimated wait times.
- Simple project structure and easy to extend.
- Development mode that automatically reloads when files change.

## Requirements

- Node.js (v18 or later recommended)
- npm (comes with Node)

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/clinic-queue-simulator.git
cd clinic-queue-simulator
```

## Install dependencies:

```bash
npm install
```

## Development

Run the project in development mode. This automatically rebuilds and restarts when code changes:

```bash
npm run dev
```

## Production Build

Compile the TypeScript source:

```bash
npm run build
```

Run the compiled output:

```bash
npm start
```
